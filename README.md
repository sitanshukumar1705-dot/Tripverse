# 🌍 TripVerse — AI Powered Travel Platform

> *"Your World, Your Journey"*

TripVerse is a comprehensive AI-powered travel platform built as a 30-Day Data Science Challenge. It provides smart trip planning, multilingual support, budget management, and vendor management for travelers across India and beyond.

---

## 🚀 Features

- 🌐 **14 Indian Languages** — Hindi, Tamil, Telugu, Marathi, Bengali, Gujarati, Kannada, Malayalam, Punjabi, Urdu, Sanskrit, Odia, Assamese, English
- 👑 **Role Based Access** — Admin, Vendor, User
- 💰 **Smart Budget System** — AI-powered budget suggestions
- 🗄️ **SQL Database** — Users, Vendors, Destinations, Bookings
- 📊 **Data Analytics** — Pandas-powered insights
- 🔐 **Secure Login** — Admin dashboard control
